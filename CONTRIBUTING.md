# Contributing to Edge Zero Starter

Thanks for contributing.

## Scope

This repository is Apache-2.0 licensed.

Contributions should focus on the starter data-plane experience:

- `blocks`: `.ez.astro` source blocks
- `cms`: Payload CMS runtime
- `types`: neutral type facade (`@edge-zero/cms-types`)
- `www`: Astro frontend runtime

## DCO Signoff

All commits should include a DCO signoff line:

`Signed-off-by: Your Name <you@example.com>`

Use:

```bash
git commit -s -m "your message"
```

## CLA for Managed Upload Surfaces

If contributions are submitted through managed Edge Zero upload flows (outside normal GitHub PR flow), CLA acceptance may be required before upload is accepted.

## Pull Request Expectations

- Keep changes focused.
- Update docs when behavior changes.
- Include tests or validation steps where practical.
